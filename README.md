# 🚀 SkillSphere AI

SkillSphere AI is an AI-powered skill development and career guidance platform designed to help students and professionals improve their technical and soft skills through personalized recommendations, assessments, and learning resources.

---

## 📌 Features

### 👤 User Management
- User Registration & Login
- Secure Authentication
- Profile Management
- Password Protection

### 📊 Dashboard
- Personalized User Dashboard
- Skill Progress Tracking
- Learning Analytics
- Performance Overview

### 🤖 AI Skill Recommendation
- AI-based Skill Suggestions
- Career Path Recommendations
- Personalized Learning Roadmaps
- Industry-Relevant Skill Mapping

### 📝 Skill Assessment

### 📚 Learning Resources

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/shiva45286/SkillSphere-Ai.git
cd SkillSphere-Ai
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows:

```bash
.venv\Scripts\activate
```

Linux/Mac:

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
